## Get Started with Astro on Vercel

Astro is an all-in-one web framework for building fast, content-focused websites. By default, Astro builds websites that have zero JavaScript runtime code.

Refer to the Astro Docs to learn more about Astro or for assistance with an Astro project.

In this guide, you will create a new Astro application and deploy it using Appz Cli.

View the [Astro Live example](https://astro.appz.dev)

### 1. Set up a new project

```bash
appz init astro
```

### 2. Install Toolchain & Dependencies

```bash
cd astro
appz setup
```
this will install dependencies, including the Nodejs runtime requirements

### 3. Start development Server

```bash
appz dev
```

### 4. Build & Preview
```bash
appz build
```
```bash
appz preview
```

### 4. Deploy
```bash
appz
```
